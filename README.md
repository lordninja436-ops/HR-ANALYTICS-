# HR-ANALYTICS
PROJECT TITLE :
HR ANALYTICS DASHBOARD : 
📌 Overview
This project showcases an interactive HR Analytics Dashboard built using Power BI. 
It provides insights into employee attrition, workforce distribution, and job satisfaction, 
helping organizations make data-driven HR decisions.

The dashboard focuses on identifying key trends and patterns in employee data 
through dynamic and interactive visualizations.

🔍 Key Insights
📊 KPI cards for quick HR insights (Employees, Attrition, Active Workforce)
📉 Department-wise and education-wise attrition analysis
👥 Employee distribution by age group and gender
😊 Job satisfaction analysis across job roles
🎯 Attrition rate breakdown by gender and age segments
🔍 Interactive filters for customized data exploration
- Overall employee count and attrition rate are clearly highlighted using KPI cards
- Highest attrition observed in specific departments (e.g., Sales/R&D)
- Majority of employees fall within the 25–44 age group
- Certain education fields show higher attrition trends
- Job satisfaction varies significantly across different roles

 ✨ Features
- Interactive KPI cards (Total Employees, Attrition Count, Attrition Rate)
- Department-wise attrition analysis
- Age group and gender-based employee distribution
- Education field-wise attrition insights
- Job satisfaction ratings by job roles
- Dynamic filters for customized analysis

 🛠 Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Cleaning & Transformation

📁 Dataset
The dataset contains HR-related information such as:
- Employee demographics (age, gender, education)
- Job roles and departments
- Attrition details
- Job satisfaction ratings
